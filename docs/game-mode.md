# Game Mode: Master the Gap

This is the guided challenge mode for the learner who wants to go from beginner to confident workflow builder.

## How to play

Start at the first rank. Do not skip ahead until the current mission is easy.

## Rank 1: Rookie Builder

**Mission:** build `ask -> save -> reply`

**Win condition:** save a user answer into `Local.Name` and return `Upper(Local.Name)`.

**Unlocks:** variable flow and local scope

## Rank 2: Branch Explorer

**Mission:** create a real if/else route

**Win condition:** branch correctly from `System.LastMessageText`.

**Unlocks:** Boolean thinking

## Rank 3: Type Detective

**Mission:** solve three workflow failures without guessing

**Win condition:** explain whether the field expected string, Boolean, table, record, blank, or message collection.

**Unlocks:** type-first debugging

## Rank 4: Message Wrangler

**Mission:** fix the agent-input gap

**Win condition:** replace failing plain text with `UserMessage(Local.PromptText)` where the field expects message-shaped input.

**Unlocks:** message collections

## Rank 5: Flow Strategist

**Mission:** route based on structured output

**Win condition:** save JSON output, read a field, and branch correctly.

**Unlocks:** records and output routing

## Rank 6: Loop Runner

**Mission:** process multiple items

**Win condition:** use a real collection in a loop without hitting a type or shape error.

**Unlocks:** tables and collection logic

## Rank 7: Workflow Guardian

**Mission:** diagnose failures fast

**Win condition:** fix `Name isn't valid`, `Type mismatch`, and a loop error by reading the field expectation first.

**Unlocks:** troubleshooting mastery

## Rank 8: Foundry Architect

**Mission:** build one end-to-end workflow

**Win condition:** combine agent input, branching, structured output, loop logic, and escalation without trial and error.
