# Foundry Workflows Power Fx Gap Guide

This repository is for the **user building Microsoft Foundry workflows in the UI** who keeps asking:

- Why did this expression fail?
- What type does this field expect?
- Why does plain text work here but not there?
- Why do I need `UserMessage(...)`?
- How do I go from beginner to confident builder without trial and error?

This repo answers those questions with a learner-focused guide for:

- fast answers
- field-to-type mapping
- a step-by-step learning path
- a question portal for missing cases

## What this guide helps with

This guide focuses on **Power Fx inside Foundry workflows**, especially:

1. field expectations
2. formula output types
3. `Local.` vs `System.`
4. strings vs Booleans vs records vs tables vs message collections
5. fixing workflow errors without guesswork
